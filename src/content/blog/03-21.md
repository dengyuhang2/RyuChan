第一步：方程整理


a x^2 + b x + c = 0

其中  a \neq 0 。

将常数项  c  移到等号右边：


a x^2 + b x = -c

第二步：配方准备

为配方，我们将二次项系数化为 1，方程两边同除以  a ：


x^2 + \frac{b}{a} x = -\frac{c}{a}

现在，左边加上一次项系数一半的平方，即  \left( \frac{b}{2a} \right)^2 = \frac{b^2}{4a^2} ，并在两边都加上它：


x^2 + \frac{b}{a} x + \frac{b^2}{4a^2} = -\frac{c}{a} + \frac{b^2}{4a^2}

第三步：写成完全平方

左边变成：


\left( x + \frac{b}{2a} \right)^2

右边通分合并：


-\frac{c}{a} + \frac{b^2}{4a^2} = \frac{-4ac + b^2}{4a^2}

记  \Delta = b^2 - 4ac ，则右边为  \frac{\Delta}{4a^2} 。

于是：


\left( x + \frac{b}{2a} \right)^2 = \frac{b^2 - 4ac}{4a^2}

第四步：开方

两边开平方，注意正负号：


x + \frac{b}{2a} = \pm \frac{\sqrt{b^2 - 4ac}}{2a}

第五步：整理出  x 


x = -\frac{b}{2a} \pm \frac{\sqrt{b^2 - 4ac}}{2a}

合并为：


x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}

最终结果


\boxed{x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}}

其中  b^2 - 4ac  称为判别式，通常用  \Delta  表示。

推导的核心步骤是配方法，将一般式化为完全平方形式，再开方求解。
